## Overview

Jodo helps communities report civic issues such as potholes, water leakages, damaged roads, overflowing garbage, broken streetlights, and public infrastructure problems.

Using Google AI, the platform automatically analyzes reports, identifies duplicates, prioritizes issues based on risk, encourages community verification, and helps authorities monitor the complete resolution lifecycle.

## Multi-Agent Architecture

| Agent                     | Responsibility                              |
| ------------------------- | ------------------------------------------- |
| Issue Intelligence Agent  | Understands and classifies civic issues     |
| Community Discovery Agent | Finds duplicate and nearby reports          |
| Verification Agent        | Validates issues through community evidence |
| Risk Intelligence Agent   | Calculates priority and severity            |
| Engagement Agent          | Coordinates community participation         |
| Resolution Agent          | Tracks issue progress and completion        |
| Analytics Agent           | Generates insights and area health metrics  |

---

## Workflow

1. Citizen submits an issue using text, image, voice, and location.
2. Gemini analyzes and classifies the report.
3. Duplicate reports are identified using embeddings and vector search.
4. Community members verify the issue.
5. AI calculates severity and priority.
6. Municipal authorities receive prioritized recommendations.
7. Resolution progress is tracked until completion.
8. Analytics are updated automatically.
